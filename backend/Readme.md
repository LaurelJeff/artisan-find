# Artisan Job Finder - Backend

Backend implementation for the Artisan Job Finder platform using Node.js and Express.

## ✅ Progress So Far

### Database & ORM
- Prisma ORM configured (v5 with @prisma/client v5)
- Database migrations set up
- User schema with initial migration
- User model extended with name field

### Authentication & Models
- User model with email and password authentication
- Database configuration setup
- Auth middleware implemented
- User controller for handling user operations

### Project Structure
- Routes configured for home and user endpoints
- Middleware layer for authentication
- Controller-based architecture for scalability
- Database configuration module

### Development Tools
- Nodemon installed globally for auto-reload
- npm dependencies installed
